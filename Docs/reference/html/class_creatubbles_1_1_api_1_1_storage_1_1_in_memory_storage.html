Detailed Description

In-memory implementation of Creatubbles.Api.Storage.ISecureStorage.

This storage class IS NOT a secure way of storing sensitive data and should ONLY be used for development and testing. For production an encrypted perstistent storage should be used instead. At the time of writing (15.11.2016) such secure storage needs to be implemented by the SDK consumer.
